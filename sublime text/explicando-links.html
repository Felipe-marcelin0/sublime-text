1. Basicamente, para se criar um link deve-se usar "a" para abrir e fechar.
    Entre os dois "a" vai ficar o nome do link.

  2. Dentro do primeiro "a" deve ser usado ' href=" endereço da pagina " '

  3. Caso queira abrir essa página em outra aba use por fim do primeiro "a" "target="_blank"

  4. A melhor forma de usar é com a "forma relativa", que é basicamente colocar "./" e mais oque for maiss importante.

    5. Aqui só foi usado ./ "nome da pesquisa".
